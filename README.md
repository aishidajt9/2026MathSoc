## 使い方

### 1. ノートブックを開く

上のバッジをクリックしてGoogle Colabでノートブックを開きます（Googleアカウントが必要です）。

### 2. ランタイムをRに切り替える

**このノートブックはRを使用します。** ColabのデフォルトはPythonなので、必ず以下の手順でRに切り替えてください。

1. メニューの「ランタイム」→「ランタイムのタイプを変更」を選択
2. 「R」を選択して「保存」

### 3. コードを実行する

各セルの左側の ▶ ボタンをクリックして、順番に実行してください。

> **注意**：Colabで開いた直後は読み取り専用です。自分で編集・保存するには「ファイル」→「ドライブにコピー」してください。
